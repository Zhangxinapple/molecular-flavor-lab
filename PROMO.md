【什么是分子风味配对？】
每种食材都有独特的"风味指纹"——由数百种挥发性芳香化合物组成。当两种食材共享关键风味分子时，它们就能产生和谐或互补的味觉体验。

【我们的引擎】
✅ 基于 FlavorDB 权威科学数据库
✅ 501种食材的完整分子指纹
✅ 590个风味标签的精确映射
✅ 两种配对模式：
   🔄 Consonance - 同味型叠加（寻找共享分子最多的搭配）
   ⚡ Contrast - 对比味型（发现互补创意的组合）

【使用场景】
🍳 家庭烹饪 - 丰富日常菜单
🍰 烘焙创作 - 探索甜点新组合
🍸 调酒实验 - 创造独特鸡尾酒

让科学指导你的烹饪创意！

## 产品亮点

### 核心功能

| 功能 | 描述 |
|------|------|
| 🔍 智能搜索 | 模糊匹配，快速定位食材 |
| 🔄 Consonance | 同味型叠加，寻找和谐搭配 |
| ⚡ Contrast | 对比味型，发现创意组合 |
| 🔬 科学解释 | 了解配对背后的分子原理 |
| 🚫 黑名单 | 排除不想看到的食材 |
| 🍳 菜谱建议 | AI 生成创意烹饪方案 |

### 数据规模

- 📊 **501** 种食材
- 🏷️ **590** 个风味标签
- 📂 **33** 个食材类别
- 🔗 **FlavorDB** 权威数据源
