# 📝 更新日志 (Changelog)

## V2.0 - 2026-02-14

### ✨ 新增功能

#### 1. 中英文双语系统
- **UI 语言切换**: 侧边栏新增语言选择器 (中文/English)
- **食材名称翻译**: 100+ 常用食材中英文映射表
- **智能搜索**: 支持中文搜索自动映射到英文数据库
- **双语显示**: 搜索结果同时显示中英文名称

#### 2. 算法可视化与评分显示
- **st.metric 分数显示**: 配对结果显著展示量化得分
- **st.progress 进度条**: 根据分数高低显示彩色进度条
- **分数等级**: 
  - ⭐⭐⭐⭐⭐ 极佳 (绿色)
  - ⭐⭐⭐⭐ 优秀 (浅绿)
  - ⭐⭐⭐ 良好 (黄色)
  - ⭐⭐ 一般 (橙色)
- **Consonance 详情**: 显示 Jaccard 相似度 + 共有风味分子数
- **Contrast 详情**: 显示对比分数 + 类别加分 + 交集加分

### 🔧 技术改进

- **路径安全**: 使用 `os.path.dirname(__file__)` 确保云端相对路径正确
- **代码注释**: 关键函数添加中文注释
- **性能优化**: 新增 `_build_name_index()` 方法加速中英文搜索

### 📁 文件变更

| 文件 | 变更 |
|------|------|
| `app.py` | 完全重写，新增双语系统和评分可视化 |
| `requirements.txt` | 保持不变 |
